# EMR-System
Electronic Medical Records (EMR) Data Analysis | Python

Built an end-to-end data analysis pipeline to ingest, clean, and structure patient data from an external API using pandas

Performed exploratory data analysis to identify trends in patient demographics and medical conditions

Created automated visualizations (bar charts, pie charts, histograms) to communicate insights clearly to non-technical stakeholders

Implemented age-based filtering to support targeted cohort analysis

Generated reproducible summary reports exporting key metrics and condition distributions

Skills demonstrated: Data cleaning, EDA, visualization, reporting, Python, pandas, matplotlib
